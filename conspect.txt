python -m venv venv - команда для створення віртуального оточення

venv/Scripts/activate - команда для активації віртуального оточення

deactivate - команда для деактивації віртуального оточення

requirements.txt - потрібен щоб швидко скачувати всі потрібні бібліотеки,
для скачування бібліотек файлу використовуємо команду pip install -r requirements.txt

django-admin startproject 'name' - команда для створення джанго проєкта

python manage.py startapp 'name' - команда для створення джанго додатка

management/commands - папка, яка відповідає за зберігання основних команд

utils - папка для зберігання додаткових файлів

__init__.py - файл, який дозволяє читати папку як модуль

git remote set-url origin 'силка git' - позволяет заменить git remote силку
